About The Tools

This page is a one stop shop for CAPE and HTAi

CAPE

CAPE uses multiple data sources to analyze and show precise analogs for a new launching drug. Compare prices and analogs across countries (currently supports USA,UK-5,Japan,China).

HTAi

HTAi leverages similar data sources to classify and predict the probability of NICE Approval in the UK. See how each parameter and attribute affects the final outcome based on the selected values.
